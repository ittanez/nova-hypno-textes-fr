# GUIDE RAPIDE DE GENERATION

## Comment utiliser ces prompts avec Nano Banana Pro

1. **Copier le prompt** de la slide souhaitee
2. **Generer l'image** avec Nano Banana Pro
3. **Ajouter le texte** incruste (soit via Nano Banana Pro s'il gere bien le texte, soit en post-production avec Canva ou un outil similaire)
4. **Verifier** que le texte est lisible sur mobile (taille minimum)
5. **Exporter en 1080x1080** pour les carrousels, 1200x1200 ou 1200x627 pour les images uniques

## Astuces pour les carrousels LinkedIn
- La **slide 1 est cruciale** : c'est elle qui donne envie de swiper
- Garder une **coherence visuelle** entre les slides (meme fond, meme police)
- La **derniere slide** doit toujours contenir le CTA et vos coordonnees
- **3 a 6 slides** est l'ideal (au-dela, le taux de completion baisse)
- Exporter en **PDF** pour uploader un carrousel sur LinkedIn

## Ordre de priorite pour la generation
Si vous manquez de temps, priorisez les visuels dans cet ordre :
1. Slides de couverture (slide 1 de chaque carrousel)
2. Images uniques des posts solo
3. Slides internes des carrousels les plus importants
4. Les slides CTA peuvent etre creees une seule fois et reutilisees
